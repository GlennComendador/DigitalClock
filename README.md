# DigitalClock
**Digital Clock Version Alpha**

This repository contains the Initial Version of the Program.
The program shows the basic Date and Time with the following
formats:

**TIME:** HH:MM:SS AM

**DAY:** Wednesday 

**DATE** Feb 02, 2022
